### Git-CLI-trial-board

###### **구토와 함께하는 git 연습장**

[레퍼런스](https://git-scm.com/book/ko/v1/%EC%8B%9C%EC%9E%91%ED%95%98%EA%B8%B0)를 참조하여 연습하도록 합시당



##### Basic : 협업 및 소스관리를 위한 기본적인 Push/Pull/Branch/Merge를 연습한다.

- git status
- git add
- git commit
- git branch
- git checkout
- git push
- git pull



##### Advance : 프로젝트를 진행하면서 생길 수 있는 여러가지 이슈들을 이해하고 대응법을 숙지한다.

- git rebase
- git cherry-pick
- git stash
- 계속 추가할 것..
